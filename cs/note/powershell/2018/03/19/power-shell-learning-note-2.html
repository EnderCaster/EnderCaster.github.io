[第二章]Windows Power Shell实战指南阅读笔记

Mar 19, 2018

摘要

本章主要介绍工具和相关显示设置

使用工具

  • Windows Powershell (x86) - 32 位powershell控制台应用程序
  • Windows Powershell ISE (x86) - 32 位powershell图形控制台应用程序 (IDE)
  • Windows Powershell - 64 位powershell控制台应用程序
  • Windows Powershell ISE - 64 位powershell图形控制台应用程序 (IDE)

注意

  • Powershell 不支持双字节字符集
  • 剪贴板操作使用的是非标准键
  • 相对于ISE，Powershell输入帮助信息较少
  • 该命令行并不是Powershell本身，只是你与Powershell的交互界面

调整外观和显示效果以适应你自己的习惯

属性(在标题栏点击右键选择属性)

  • 缓冲区用于调整可以选择的命令历史长度
  • 适当提高字体大小有利于保护视力，以能够清楚地分清重音符号和单引号为准
  • 颜色建议不修改，高反差颜色配置可以有效地提高识别准确率
  • tab 键补全是很好用的一个功能，在ISE中体现为智能提示

几点建议

  • 配置合适的大小，使窗口不出现滚动条是很重要的
  • 强烈建议使用64位操作系统下的64位Powershell
  • 为了避免一些问题，在实验阶段尽量保证运行在管理员权限下

题目

本章课后练习没有太好的方式做题解，故此处只引题目

  • 选择适合你的字体和颜色
  • 确保控制台程序下方没有水平滚动条
  • 在ISE中最大化控制台窗格，移除或最小化命令管理器
  • 在所有应用程序中，输入一个单引号和一个重音符号，调整字体以便你能很好的识别这些符号
  • 同样输入括号、中括号、尖括号和大括号，并进行检验
